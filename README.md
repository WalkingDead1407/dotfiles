# My Dotfiles

Personal dotfile configurations for:
- Btop (system monitor)
- Kitty (terminal emulator)
- Neovim (text editor)
- Pokemon-fetch (system info)
- Lazygit (git UI)
- Bash (shell)

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
source ~/.bashrc
```

## Files Included

- `.config/kitty/` - Kitty terminal configuration
- `.config/nvim/` - Neovim configuration
- `.config/pokemon-fetch/` - Pokemon-fetch configuration
- `.config/lazygit/` - Lazygit configuration

## Usage

All configs are managed through symlinks. Changes made to files in this repo are reflected immediately.

Enjoy your customized development environment! 🎉
